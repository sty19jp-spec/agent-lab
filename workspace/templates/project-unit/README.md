# Project Unit Template

This directory is the canonical Project Unit Template for Track B.
It is intended to be copied into a future project workspace.

This is a B-phase template implementation: human-readable structure first, machine-readable template definition later.

## Top-level directories

- `pipeline`: execution design for the project unit. It contains `steps` and `workflows`.
- `distribution`: channel-specific publishing structure. It contains `youtube`, `instagram`, `x`, `note`, and `blog`.
- `inputs`: source materials and intake files used to start the content flow.
- `assets`: reusable media, reference assets, and supporting materials.
- `outputs`: generated deliverables and packaged results.
- `analytics`: reporting, measurement, and performance tracking artifacts.
- `config`: project-local configuration and operating settings.
